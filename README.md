# Best Shop - travel suitcase webshop development (HTML/SCSS/JS)

Best Shop front-end application built with Vite, Sass, ESLint, and Stylelint.

## Overview

This repository contains the implemented front-end for the **Best Shop** project.  
It is based on a capstone starter setup, but now includes latest code and customizations.

Main technologies:

- Development server powered by **Vite**
- Styling with **SCSS/Sass**
- **ESLint** for JavaScript linting
- **Stylelint** for SCSS linting
- Predefined npm scripts to streamline development

---

## Prerequisites

Make sure you have:

- **Node.js** (LTS version recommended)
- **npm** (comes with Node.js)

You can verify your versions with:

```bash
node -v
npm -v
